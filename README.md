# Heart Disease Risk Prediction

Predicting heart disease risk from clinical data using machine learning.

## Dataset
UCI Heart Disease Dataset via Kaggle.

## Project Structure
- `notebooks/` — Jupyter notebooks for EDA and modeling
- `data/` — raw dataset (not tracked by Git)

## Tools
Python, pandas, scikit-learn, XGBoost, seaborn, SHAP# heart-disease-prediction
